About Us

Welcome to the Pixabay Image Search app. This web app allows you to search for high-quality images using the Pixabay API. You can search by keyword and category to find the images you need.

Our mission is to make finding and using high-quality images as easy and efficient as possible. We hope you enjoy using our app!
